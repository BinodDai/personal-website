## ğŸ“¤ GitHub Deployment

### Step 1: Create Repository
1. Go to [GitHub](https://github.com)
2. Click "New Repository"
3. Name it: `portfolio` or `binod-basnet.com.np`
4. Make it **Public**
5. Click "Create repository"

### Step 2: Upload Files
1. Click "uploading an existing file"
2. Drag and drop these 3 files:
   - `index.html`
   - `profile.jpg`
   - `Binod_Basnet_CV.pdf`
3. Click "Commit changes"

### Step 3: Enable GitHub Pages
1. Go to repository "Settings"
2. Click "Pages" in left sidebar
3. Under "Source", select "Deploy from a branch"
4. Select branch: `main` or `master`
5. Select folder: `/ (root)`
6. Click "Save"

Your site will be live at: `https://yourusername.github.io/portfolio`

### Step 4: Connect Custom Domain (binod-basnet.com.np)

#### In GitHub:
1. Go to Settings â†’ Pages
2. Under "Custom domain", enter: `binod-basnet.com.np`
3. Click "Save"
4. Check "Enforce HTTPS"

### Step 5: Wait for DNS
- DNS changes take 24-48 hours
- GitHub Pages will issue SSL certificate automatically
- Your site will be live at `https://binod-basnet.com.np`
